# Snap App

A Vite + TypeScript Progressive Web App for the interactive Snap button experience.

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run preview` - preview production build

## PWA

- Service worker registration is in `src/main.ts`
- Manifest and SW generation are configured in `vite.config.ts`

## Run

1. `npm install`
2. `npm run dev`
3. `npm run build`
